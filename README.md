## Installation

Follow the steps below to get started with **office_administration**:

1. Get the app (repo URL is required):

    ```bash
    ./3plug get-app <repo_url> office_administration --bundle <bundle_name>
    ```

2. Install in site:

    ```bash
    ./3plug install-app --site [sitename] office_administration
    ```

3. Install dependencies:

    ```bash
    ./3plug install
    ```

4. Migrate the site:

    ```bash
    ./3plug migrate --site [sitename]
    ```
